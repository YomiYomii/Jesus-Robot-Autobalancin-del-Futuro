# Lista de Materiales (BOM)
 
Lista de componentes electrónicos y materiales utilizados en la fabricación del robot auto-balancín.

## Componentes electrónicos
 
| Componente | Cantidad | Función |
|---|---|---|
| Motor DC 6V | 2 | Tracción de las ruedas del robot |
| Sensor/giroscopio MPU6050 | 1 | Medición del ángulo de inclinación (IMU) |
| Driver de motores L298N | 1 | Control de potencia y sentido de giro de los motores |
| Arduino (Uno/Nano)* | 1 | Controlador principal, ejecuta el algoritmo PID |
| LED amarillo | 6 | Iluminación decorativa (cruz estética) |
| Portapilas para 2 baterías 12V | 1 | Alimentación eléctrica del sistema |
| Rueda 60 mm | 2 | Ruedas motrices |
| Breadboard parcial | 1 | Conexión y montaje de componentes electrónicos |
 
\* *Ajusta el modelo exacto de Arduino usado (Uno, Nano, Mini, etc.)*
